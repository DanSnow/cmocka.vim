cmocka.vim

## Description ##
CMocka is a C unit testing framework. And this is syntax highlight for cmocka.

## Installation ##
- Vundle
```
Plugin 'DanSnow/cmocka.vim'
```
- vim-plug
```
Plug 'DanSnow/cmocka.vim'
```
